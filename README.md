# 대한항공 홈페이지 퍼블리싱 연습
리엑트 퍼블리싱 연습 <br/>
[대한항공 홈페이지](https://www.koreanair.com/)를 참고하여 퍼블리싱 연습 <br/>
<br/>

## 시작
```
npm i
npm start
```
<br/>
