# 대한항공 홈페이지 클론
[대한항공 홈페이지](https://www.koreanair.com/)의 메인페이지 디자인을 참고하여 클론한 페이지 입니다. <br/>
React와 Styled-Components를 사용하여 웹페이지 구현속도를 측정하기 위한 목적으로 진행하였습니다.


<br/>

## 시작
```
npm i
npm start
```
<br/>

## 사용 스택

`React` `Styled-Components` <br/>
`Javascript` `CSS` `HTML`

<br/>

## 기간
* 22.09.19(월), 22.09.21(수) ~ 22.09.29(목) (주말 제외 총 8일)
